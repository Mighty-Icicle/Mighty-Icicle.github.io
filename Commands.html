Useful commands for csgo

Main page

Use a command by pressing the ~ key (top left on your keyboard) and entering it into the console

god
Grants godmode and makes you invincible

sv_infinite ammo 1
Gives the player infinite ammo, can be 0,1, or 2.
0 is normal
1 is infinite ammo without reload
2 is infinite ammo with reload

noclip
Allows the player to pass through any object, you are invincible once you have done this.

r_drawothermodels 2
Renders player models as wireframe allowing you to see them through walls

mat_wireframe 2
Renders models and props as wireframe allowing you to see through walls. Can be 1,2,3, or 4. Default is 0.

mat_fullbright 3
Renders walls and models white. Can be 1,2, or 3. Default is 0

mat_fillrate 1
Allows you to see even if you are fully flashed

r_drawparticles 0
Doesnt draw smoke particles

thirdperson or firstperson
Puts you in thirdperson mode or firstperson mode

BindToggle *Key* cl_righthand
Will swap the side on which you hold things when you press the button.

snd_show + snd_visualize 1
Shows information about the sounds.

cl_leveloverview 2
Shows a level overview (kind of buggy)
Can be 0, 1, or 2.

host_timescale X
Replace X with a number.
Use 2 for super speed and 0.5 for slow motion

host_framerate X
Replace X with a number
Use 50 for super speed and 500 for slowmotion, default is 0.

cl_pitchup 900 & cl_pitchdown 900
Unlocks max up and max down pitch
Default if 89

r_showenvcubemap 1
Makes all models chrome

mat_showlowresimage 1
Creates an extra low-resolution version of csgo, can improve performance.
